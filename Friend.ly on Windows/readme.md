## Friend.ly on Windows
We’re really excited to share that we’re in the process of developing the Windows version of Friend.ly! This update means you’ll soon be able to enjoy the full Friend.ly experience on your Windows PC, with a design that’s fast, reliable, and optimized for the Windows environment. We’re focusing on a clean, responsive, and intuitive interface, built using modern Windows development tools to deliver strong performance and a seamless desktop experience.
